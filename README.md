# TramApp

TramApp is an intelligent Android application for tracking nearby tram schedules, prioritizing favorite stations, and providing smart routing logic to key destinations (Home, Work, School).

## Features
- Real-time departure boards (Powered by Golemio API)
- Intelligent destination highlighting
- Favorite stations and lines
- Offline caching and graceful degradation

## Tech Stack
- Kotlin
- Jetpack Compose
- Hilt (Dependency Injection)
- Room Database (Local Caching)
- Retrofit & OkHttp (Networking)
